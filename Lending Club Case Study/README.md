# Lending-Club-Case-Study
To identify variables which are strong indicators of default and potentially use the insights in approval / rejection decision making for lending club

# Description
Provided a Data Intake Report, Count of Records, and Assumed Schema.Created an EDA Report showing basic correlations and distributions of data and possible feature engineering suggestions.
The data contains missing values. After preprocessing the data and performing feature engineering, I have implemented three regression models namely Logistic Regression, Random Forest and SVM.Logistic Regression has better performance over Random Forest and SVM with a ROC AUC of 0.67 on test set
